# bearing-fault-diagnosis

## Instructions for using the PHM dataset

To utilize the PHM dataset in this framework, follow the instructions below:

1. **Download the PHM dataset** from the PHM 2024 website https://www.icphm.org/.
2. **Extract the downloaded data** into the root folder of this framework.
3. **Open the `save_phm.py` file** located in the `data_loaders` folder.
4. **Execute the `save_phm.py` script**. This script will generate a directory named `raw_phm` containing all the data with the appropriate structure for conducting experiments within the framework.
